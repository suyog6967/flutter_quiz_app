Flutter Quiz App
A interactive Quiz Application built with Flutter. This app tests the user's knowledge about Flutter internals, tracks their score, and provides a detailed summary of correct and incorrect answers at the end.

Interactive UI: Smooth navigation between the Start, Questions, and Results screens.

Dynamic State Management: Uses setState to manage the quiz flow and user progress.

Custom Styling:

Beautiful Linear Gradient background.

Custom buttons using ElevatedButton and OutlinedButton with specific styling.

Google Fonts integration (Oswald and Lato) for modern typography.

Score Tracking: Calculates the final score based on user input.

Detailed Summary: The Results Screen displays a scrollable list showing the question, the user's answer, and the correct answer, with color-coded indicators.

Restart Functionality: Allows the user to reset the quiz and try again.
